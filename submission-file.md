## Introduction to Application

**Focus Sphere** is an innovative, gamified Pomodoro application designed to revolutionize traditional time management by integrating the concept of the 'Connected Environment'. Unlike standard productivity tools that solely track time, Focus Sphere utilizes the device's native hardware sensors (ambient light and microphone) to actively monitor the user's physical surroundings. 

The core gamified narrative revolves around 'charging the Focus Planet'. When the environment is optimal (quiet and well-lit), the planet charges successfully. However, if the ambient light drops below 300 Lux or the noise exceeds 60 dB, the app triggers a visual 'Energy Crisis' alert, physically prompting the user to improve their workspace. 

Furthermore, the application features seamless cloud integration via Firebase Cloud Firestore. It records real-time environmental data arrays during active sessions, allowing users to upload, manage, and review their study history. This data is visually translated into interactive line charts using `fl_chart`, providing users with actionable insights into their productivity patterns and environmental health.
